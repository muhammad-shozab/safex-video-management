# Screenshots

Add your project screenshots to this folder.

The main `README.md` expects these image files:

| File Name | Where to Use |
|-----------|--------------|
| `logo.png` | Top of README, project logo |
| `dashboard.png` | Admin Dashboard section |
| `search.png` | YouTube Video Search section |
| `library.png` | Video Library section |
| `review.png` | Review Workflow section |
| `users.png` | User & Role Management section |
| `login.png` | Login / Signup section |

## How to add screenshots

1. Run the app locally (`npm run dev` in frontend, `dotnet run` in backend).
2. Open each page in your browser.
3. Press `Windows + Shift + S` (Windows) or `Cmd + Shift + 4` (Mac) to take a screenshot.
4. Save the image into this folder with the correct file name.
5. After adding images, push them to GitHub:

```bash
git add screenshots/
git commit -m "Added project screenshots"
git push
```

## Tips

- Use PNG or JPG format.
- Keep image widths around 1200px for best quality on GitHub.
- Make sure the app looks clean before taking screenshots.
- Use the same browser window size for consistency.
